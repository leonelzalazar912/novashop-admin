# NovaShop

NovaShop es una plataforma base para crear tiendas online personalizadas para pequeños y medianos comercios.

## Objetivo

Convertir un prototipo de e-commerce en un software adaptable, reutilizable y comercializable.

## Público objetivo

- Tiendas de ropa
- Tiendas gamer
- Perfumerías
- Librerías
- Ferreterías
- Casas de informática
- Comercios locales

## Problema que resuelve

Muchos comercios venden por Instagram o WhatsApp, pero no tienen una tienda online profesional con catálogo, carrito y gestión de pedidos.

## Propuesta de valor

NovaShop permite crear una tienda online personalizada, rápida y profesional, adaptable al rubro del cliente.

## Funcionalidades iniciales

- Home comercial
- Catálogo de productos
- Buscador
- Filtros
- Detalle de producto
- Carrito
- Control de stock
- Checkout
- Configuración de marca

## Futuras funcionalidades

- Panel administrador
- Base de datos
- Gestión de productos
- Gestión de pedidos
- Mercado Pago
- WhatsApp integrado
- Múltiples plantillas por rubro

## Fases del proyecto

### Fase 1 - Convertir el prototipo en e-commerce genérico

- Crear storeConfig
- Cambiar Game por Product
- Cambiar allGames por products
- Sacar textos exclusivos de videojuegos
- Hacer filtros genéricos

### Fase 2 - Panel administrador

- Login admin
- Crear productos
- Editar productos
- Cambiar stock
- Ver pedidos

### Fase 3 - Base de datos

- Conectar Supabase
- Guardar productos reales
- Guardar pedidos reales
- Gestionar stock real

### Fase 4 - Venta real

- Checkout funcional
- WhatsApp
- Mercado Pago
- Confirmación de pedido

### Fase 5 - Producto comercial

- Demo para distintos rubros
- Presentación comercial
- Planes de precios
- Oferta para comercios reales
